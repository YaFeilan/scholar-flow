## üìñ Introduction

**ScholarFlow** is an intelligent academic platform designed to cover the entire research lifecycle. Powered by **Gemini**, it streamlines workflows from topic selection and data analysis to paper writing and submission.

The project is built with **TypeScript** for robust type safety and integrates advanced visualization tools (D3.js) and PDF export capabilities.

## ‚ú® Features

Comprehensive tools covering every step of your research journey.

### üî¨ 1. Research Planning
* **Topic Guide:** AI-driven brainstorming to generate and validate research ideas.
* **Research Discussion:** Interactive chat interface to refine academic concepts.
* **Trend Analysis:** Visualize emerging topics and academic trends.
* **Academic Search:** Integrated search engine for scholarly papers.
* **Literature Reading:** Deep reading assistant with summarization capabilities.
* **Knowledge Graph:** Visual mapping of connections between concepts and papers.
* **Proposal Review:** AI evaluation of research proposals.
* **Grant Application:** Assistance in drafting and refining grant proposals.

### üìä 2. Experiments & Data
* **Experiment Design:** Plan robust experimental protocols and methodologies.
* **Data Analysis:** Automated statistical analysis and insight generation.
* **Chart Extraction:** Extract raw data from existing paper figures.
* **Code Assistant:** Generate, debug, and optimize research code (Python/R, etc.).
* **Literature Tracking:** Stay updated with the latest publications in your field.

### ‚úçÔ∏è 3. Paper Writing
* **Review Generation:** Automatically generate literature reviews based on selected papers.
* **Polishing Assistant:** Enhance academic tone, grammar, and clarity.
* **Scientific Plotting:** Create publication-ready visualizations and diagrams.
* **PPT Generation:** Convert research papers into presentation slides instantly.

### üöÄ 4. Submission & Publication
* **Submission Advisory:** Analyze manuscript content to recommend suitable journals.
* **Peer Review Sim:** Simulate peer review feedback to identify potential weaknesses.
* **Call for Papers:** Aggregate latest CFP information from conferences and journals.
* **AI Detection:** Detect AI-generated content patterns to ensure originality.

## ‚ö° Deployment / Local Preview

Follow these steps to build and preview the project locally.

1.  **Get the Code:**
    Clone the repository or download the source code to your local machine.

2.  **Install Dependencies:**
    ```bash
    npm install
    ```

3.  **Configure Environment:**
    Open the existing `.env.local` file in the project root directory and replace the placeholder with your actual API key:
    ```env
    GEMINI_API_KEY=your_actual_api_key_here
    ```

4.  **Build for Production:**
    Compile the project for production deployment:
    ```bash
    npm run build
    ```

5.  **Run Preview:**
    Launch a local server to preview the built project:
    ```bash
    npm run preview
    ```
    Once running, open your browser and visit the default address (usually `http://localhost:4173`) to view the deployed application.
